# ランダムグッズ購入支援ツール

## 概要

複数のグループに分かれたランダムグッズの購入計画をサポートするツールです。

どのグループを何個購入すれば、ほしいキャラクターを効率よく狙えるかを入力した評価値をもとに計算します

## 使用方法

1. 作品を選択
2. 各キャラクターを1~10で評価
3. 購入予定総個数・当たり率（どこから当たりか）を設定
4. 「おすすめを計算」を押す

## 評価の目安
・10:最推しまたは超当たり。最優先で欲しい
・8~9:当たり。できれば欲しい
・5～7:普通。当たれば嬉しい
・2～4:あまり必要ではない
・1:不要
・0:未評価

## 評価データの保存
入力した評価はJSONファイルとして保存できます
次回は保存したファイルを読み込むことで、前回の評価をそのまま利用できます
